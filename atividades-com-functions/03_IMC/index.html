Atividade 3:

Utilizando função: Tendo como dados de entrada o peso (em quilogramas) e a altura (em metros) de uma pessoa, crie um programa que calcule o Índice de Massa Corporal (IMC) dessa pessoa.

Calcule o IMC usando a fórmula: IMC = peso / (altura * altura) e classifique o IMC da seguinte forma:

  • IMC < 18,5 Kg/m²: Abaixo do Peso

  • IMC >= 18,5 Kg/m² e < 24,9 Kg/m²: Peso Ideal

  • IMC >= 25 Kg/m² e < 29,9 Kg/m²: Excesso de Peso

  • IMC >= 30 Kg/m²: Obesidade

Abra o console para verificar o programa.
